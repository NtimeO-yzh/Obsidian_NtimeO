# 系统自带的快捷键

| 动作       | 快捷键        |
| ---------- | ------------- |
| 复制       | `cmd+C`       |
| 剪切       | `cmd+X`       |
| 粘贴       | `cmd+V`       |
| 无格式粘贴 | `cmd+Shift+V` |
| 撤销       | `cmd+Z`       |
| 重做       | `cmd+Y`       |
| 查找(当前) | `cmd+F`       |
| 全局查找   | `cmd+Shift+F` |
| 替换(当前) | `cmd+H`       |
| 插入链接   | `cmd+k`       |
| 字体加粗   | `cmd+B`       |
| 切换“阅览”和“编辑”模式           |  `cmd+e`             |

| 打开右侧边栏 | `cmd+⬅️` |
